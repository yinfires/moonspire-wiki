# Moon Spire 项目总览

Moon Spire 是一个基于 NeoForge 的 Minecraft 1.21.1 模组。它把牌组构筑、回合制卡牌战斗和爬塔路线选择带进 Minecraft 世界。

## 模组定位

玩家可以挑战生物、击败怪物获取卡牌、消耗费用打出卡牌，并管理抽牌堆、弃牌堆和被消耗的牌。

模组同时提供爬塔流程、塔内专用卡组、制卡台转换装备、怪物意图、状态效果、战斗动画反馈，以及用于自定义内容的游戏内工具。

## 玩法关键词

- **卡牌战斗**：战斗单位拥有费用、手牌、抽牌堆、弃牌堆和被消耗牌堆。
- **装备转牌**：制卡台可以把装备转换成可在战斗中使用的卡牌。
- **爬塔跑局**：玩家进入 `moonspire:tower` 维度后，在路线图中选择战斗房或篝火房。
- **塔内隔离**：塔内会替换为塔内卡组，并用备份机制恢复进入前物品、状态和正常卡组。
- **创作者工具**：开启开发者模式并拥有管理员权限后，可以在游戏内调整卡牌、卡面、怪物、房间、怪物池、塔配置和战斗 UI。
