# 创作者工具

Moon Spire 提供一组游戏内创作者工具，方便管理员调整卡牌、怪物、房间和爬塔配置。这些工具主要服务于地图制作、玩法调试和自定义内容创作，普通游玩不需要打开。

## 开启条件

使用创作者工具前，需要先在 Moon Spire 的客户端配置文件中开启开发者模式。进入世界后，操作者还需要拥有 2 级或更高管理员权限。

如果没有开启开发者模式，相关按键入口不会显示；如果没有管理员权限，即使打开了界面，也无法执行房间编辑、爬塔调试或内容保存等操作。

## 开发者中心

开发者中心集中管理 Moon Spire 的可编辑内容。它包含以下页面：

| 页面 | 可以调整的内容 |
| --- | --- |
| 卡牌 | 自定义卡牌名称、费用、卡图、手持物和卡牌效果 |
| 卡面 | 卡面图片，以及费用、名称、卡图、类型、描述区域 |
| 怪物 | 怪物生命、费用、速度、初始状态、卡组和奖励池 |
| 房间 | 爬塔房间名称、类型，以及战斗房使用的怪物池 |
| 塔配置 | 楼层数量、塔生命上限、路线节点、房间类型和候选池 |
| 怪物池 | 怪物组、权重和组内实体 |
| 战斗 UI | 战斗界面元素的位置和尺寸 |

这些页面适合用来快速搭建一套自定义爬塔内容，例如先创建房间，再配置怪物池，最后把它们放进塔配置里。

## 房间编辑

房间编辑工具用于在专用编辑场景中搭建爬塔房间。工具可以记录房间范围、玩家出生点、怪物出生点和触发范围。

颜色约定：

| 编辑内容 | 边框颜色 |
| --- | --- |
| 房间范围 | 蓝色 |
| 玩家出生点 | 绿色 |
| 怪物出生点 | 黄色 |
| 触发范围 | 红色 |

战斗房需要设置触发范围和怪物池。篝火房需要在房间内放置月尖塔篝火。

## 怪物池

怪物池由多个怪物组组成。跑局抽中某个组时，组内怪物会一起加入战斗。

同一个怪物组里可以放入重复怪物，用来制造更密集的遭遇。空组不会被抽中；无法使用的实体会在运行时跳过，因此适合先保存草稿，再逐步补齐。

## 塔配置

塔配置决定一次爬塔的结构。每层可以设置房间数量、每一步可选节点数、战斗与篝火权重、固定节点，以及允许出现的房间和怪物池。

如果某层没有单独设置规则，会沿用第一层的配置。这样可以先做一套通用路线，再给关键楼层补特殊规则。

## 卡牌与怪物

卡牌页适合制作新的可用卡牌或调整已有自定义卡牌。怪物页适合给某个实体指定专属卡组、奖励池和初始状态。

怪物奖励一般来自它进入战斗时使用的卡组。为怪物单独设置奖励池后，玩家击败它时就可以从指定卡牌里获得奖励。
