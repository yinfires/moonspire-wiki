# 登塔之旅

登塔之旅是一局独立的爬塔冒险。进入塔后，玩家会使用本局专用卡组和登塔生命，在路线图上选择战斗房或篝火房，逐步推进到终点。

## 建立入口

玩家可以像点燃下界传送门一样，用黄色混凝土搭出竖直矩形外框，再用打火石点燃框内空间生成月尖塔传送门。

| 结构 | 最小尺寸 |
| --- | --- |
| 外框 | 4x5 |
| 内部空间 | 2x3 |

传送门可以做得更大，最大尺寸接近原版传送门。通过黄色混凝土门框生成的传送门会绑定门框；拆掉门体或外框会让整片传送门破碎。玩家从外侧进入塔后，即使外侧传送门之后被破坏，离开塔时仍会回到进入时记录的位置。

## 准备大厅

走入月尖塔传送门会进入登塔准备大厅。准备大厅会暂时保存进入前的背包、状态和常规卡组，并清空当前物品栏，防止外部资源影响本局冒险。

站在准备大厅的开始范围内时，界面会提示按 `N` 开始登塔之旅。多人一起在开始范围内时，会作为同一队进入本次跑局。准备大厅中的传送门也可以用于离开大厅并恢复进入前状态。

管理员也可以用登塔相关指令作为调试入口，但普通游玩推荐通过月尖塔传送门和准备大厅开始。

## 路线地图

默认塔配置包含 `80` 点登塔生命和 `12` 个路线房间。地图按自下而上的方向显示路线，第一个选择在底部，后续房间向上延伸。

当前路线节点分为两类：

| 节点 | 作用 |
| --- | --- |
| 战斗 | 进入房间触发区后开始战斗 |
| 篝火 | 靠近月尖塔篝火恢复登塔生命 |

路线图只显示节点类型、连线和当前可选状态，不会提前透露具体怪物。选中节点后，队伍会进入对应房间；完成房间后，地图会再次打开并显示下一步可选节点。

## 战斗房

战斗房需要玩家进入触发区后才会开战。战斗开始时，怪物会从该房间的怪物池中抽取并加入战斗。

胜利后，玩家可以继续领取奖励并推进路线。塔内获得的卡牌会加入本次登塔使用的塔内卡组，而不是立即写入进入塔前的常规卡组。

## 篝火房

篝火房中的月尖塔篝火会在玩家靠近时恢复 `10` 点登塔生命，最高不会超过本局登塔生命上限。同一名玩家在同一个篝火房内只能恢复一次。

当所有在场玩家都完成恢复后，篝火会熄灭，房间完成，路线地图会重新打开。

## 结束与恢复

登塔完成、失败、放弃或从大厅离开时，系统会恢复进入塔前保存的物品、状态、游戏模式和常规卡组。

如果队伍是从准备大厅正式开局，跑局结束后会先回到准备大厅；再次走入大厅传送门离开时，才会恢复进入塔前的状态。这样玩家可以清楚地区分“本局已结束”和“已经离开月尖塔空间”。
