# 界面与反馈

Moon Spire 的战斗界面会把牌组、费用、状态和敌人意图集中显示出来，让玩家在 Minecraft 世界里完成一场完整的卡牌战斗。

## 战斗界面

战斗开始后，玩家会看到手牌、费用、抽牌堆、弃牌堆和被消耗牌入口。每张卡牌会显示费用、名称、类型、卡图和效果描述。

常见界面元素：

| 元素 | 作用 |
| --- | --- |
| 手牌 | 当前可以打出的卡牌 |
| 费用 | 打出卡牌需要消耗的资源 |
| 抽牌堆 | 回合开始时从这里抽牌 |
| 弃牌堆 | 已打出或回合结束丢弃的牌会进入这里 |
| 被消耗牌 | 本场战斗结束前暂时离开牌组的牌 |
| 结束回合 | 结束当前玩家回合并推进战斗 |

## 单位信息

玩家和敌人条目会显示生命、格挡、速度、状态效果和意图。速度会影响部分伤害预览；格挡会优先吸收伤害；状态图标会展示层数和说明。

敌人的意图会提示它接下来可能进行攻击、防御或施加状态。阅读意图后安排出牌，是 Moon Spire 战斗里的重要决策点。

## 卡牌预览

悬停卡牌时，可以查看放大预览和关键词说明。卡牌描述中的关键词会高亮，并在旁边展示解释，例如“消耗”“保留”“远程”“箭”等。

攻击数值会根据当前力量、虚弱、速度、格挡和其它状态进行预览。数值增强或削弱时，会用不同颜色帮助玩家判断这张牌当前是否值得打出。

## 出牌动画

打出卡牌后，界面会播放卡牌飞出或消失的反馈。不同类型的牌会触发不同世界表现：

| 表现 | 示例 |
| --- | --- |
| 近战突进 | 攻击者冲向目标并打击 |
| 远程射击 | 弓、弩或箭类牌发射弹射物 |
| 药水投掷 | 药水飞向目标并在命中点破裂 |
| 光束与法术 | 守卫者光束、唤魔者尖牙等特殊表现 |
| 践踏冲击 | 特定怪物牌命中时出现贴地冲击波 |

## 世界反馈

造成伤害、获得格挡、恢复生命或改变状态层数时，目标附近会出现数字和图标反馈。状态增加显示正数，状态减少显示负数，帮助玩家追踪每次结算的结果。

战斗结束后，敌人会恢复到安全位置，玩家则根据战斗结果继续领取奖励、返回路线图或离开当前跑局。
