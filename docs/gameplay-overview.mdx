# 玩法概览

这一页展示玩家能直接体验到的玩法：战斗怎样进行、牌堆如何运作、装备如何转成卡牌，以及登塔之旅如何接入这些系统。

## 战斗流程

Moon Spire 的核心战斗是牌组构筑式回合战斗。玩家在自己的回合消耗费用打出手牌，敌人会根据自己的怪物卡组和意图行动。界面中会显示玩家与敌人的生命、格挡、速度、状态效果、抽牌堆、弃牌堆和被消耗牌。

战斗中的伤害、格挡、治疗和状态层数变化会通过卡牌动画、世界飘字、状态图标和实体动作反馈出来。近战突进、弓弩射击、药水投掷、三叉戟、火球、光束和潜影弹等表现会根据卡牌类型呈现不同效果。

## 卡牌与牌堆

每场战斗都会维护抽牌堆、弃牌堆、手牌和被消耗牌。回合开始时抽牌，出牌后通常进入弃牌堆，带有“消耗”关键词的牌会在本场战斗结束前离开牌组。

游戏中常见关键词与状态包括：

| 类型 | 示例 |
| --- | --- |
| 关键词 | 费用、格挡、消耗、固有、保留、虚无、远程、箭 |
| 状态 | 流血、凝视、守护、不死、召唤物、分裂、力量、再生、迅捷、相位 |
| 负面状态 | 中毒、烧伤、凋零、虚弱、缓慢、黑暗、饥饿、潮蚀、麻痹 |

## 制卡台

制卡台用于把装备或特定物品转换为卡牌。玩家点击可转换装备后，会消耗该物品并生成对应卡牌。安装 JEI 时，也可以在“制卡台转换”分类中查看输入物品和输出卡牌。

弓、弩、箭和光灵箭有专用转换牌：

| 物品 | 转换效果摘要 |
| --- | --- |
| 弓 | 远程，消耗一根箭并造成伤害 |
| 弩 | 远程，消耗一根箭并造成更高伤害，带保留与保留降费 |
| 箭 | 箭，造成基础伤害并消耗 |
| 光灵箭 | 箭，造成基础伤害并给予发光 |

## 登塔之旅

玩家可以用黄色混凝土搭出月尖塔传送门外框，再用打火石点燃框内空间生成月尖塔传送门。走入传送门会进入准备大厅，在开始范围内按 `N` 开始登塔之旅。

进入登塔后，玩家会在路线图上选择下一间房，并围绕登塔生命、塔内卡组和房间奖励推进跑局。默认塔有 `80` 点登塔生命和 `12` 个路线房间，路线图自下而上显示。

当前登塔规则支持战斗房与篝火房。地图阶段只显示节点类型和可选状态，不提前透露具体怪物内容。战斗房进入触发区后开战；篝火房靠近月尖塔篝火会恢复 `10` 点登塔生命，同一玩家同一篝火房在本次跑局只恢复一次。

更完整的入口、准备大厅、路线选择和恢复规则见 [登塔之旅](./tower-journey)。

## 塔内隔离与恢复

进入塔时，玩家会暂时使用塔内规则和塔内卡组，原本的物品与常规卡组会在离开登塔后恢复。

跑局完成、失败、放弃或离开准备大厅时，玩家会回到进入前的状态。塔内奖励会加入本次跑局使用的塔内卡组，让路线推进更像一次独立冒险。
